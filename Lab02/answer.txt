10. If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?

Yes, Java allows you to create multiple constructors in a class. This concept is called constructor overloading.

You can define multiple constructors in a class as long as they have different parameter lists (different number or types of parameters). Java determines which constructor to use based on the arguments provided when creating an object.


15. Is JAVA a Pass by Value or a Pass by Reference programming language?

Java is strictly pass-by-value because it always passes copies of variables to methods, the confusion arises because Java passes copies of references when dealing with objects, making it seem like pass-by-reference.

However, the way Java handles object references can sometimes create confusion, making it seem like pass-by-reference.
